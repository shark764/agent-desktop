## Quick start

1. Clone this repo using `git clone git@github.com:liveops/agent-desktop.git`
2. install yarn globally `npm -g install yarn`
3. `cd agent-desktop`
4. run `yarn`
5. run `npm start`

Now you're ready to rumble!
